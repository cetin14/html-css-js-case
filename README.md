# Nexcent - Modern Landing Page

This project is a fully **responsive (mobile-friendly)** landing page built using **HTML5**, **CSS3**, and the **BEM (Block Element Modifier)** methodology. 

It was developed as a clean-code interface study to practice modern web development layouts (Flexbox & Grid), scalable styling architectures, and responsive design principles.

## Features

- **BEM Methodology:** Clean, scalable, and highly maintainable CSS structure.
- **CSS Custom Properties (Variables):** Centralized management for the color palette, typography scaling, line heights, and shadows.
- **Fully Responsive Layout:** Seamless adaptation to desktop, tablet, and mobile screens utilizing tailored media queries (992px and 768px breakpoints).
- **Swiper JS Integration:** A dynamic and modern touch-enabled carousel framework powering the Hero section.
- **Modern UI Components:** Advanced interface layouts including modern blog cards overlapping imagery via negative margins, interactive button states, and dynamic brand logos utilizing CSS `grayscale` filters.

## Tech Stack

* **HTML5:** Semantic markup structure.
* **CSS3:** Flexbox, Grid Layout, Custom Properties, Media Queries.
* **Google Fonts:** Utilizing the 'Inter' font family for slick typography.
* **Swiper JS:** Slider/Carousel mechanism.
